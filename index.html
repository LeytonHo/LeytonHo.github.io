NASA: System Health Enabled Realtime Planning Advisor

I contributed to a decision making support framework for planetary rover exploration missions, accounting for uncertainty with POMDPs. I focused on building a new, extensible traverse synthesis use case to fully automate a rover’s path according to mission goals (represented by a reward function).

GrowCut-based Melanoma Lesion Segmentation

I designed and implemented a fully-automated skin lesion segmentation pipeline to improve melanoma diagnosis. The pipeline included hue and saturation color thresholding using three-fold validation, connected component analysis, and refined boundary delineation using a cellular automaton technique.
